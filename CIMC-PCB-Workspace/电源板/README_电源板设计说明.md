# 电源板设计说明（基于GD30DC1354）

## 一、设计目标

将工业标准24V DC输入转换为5V DC输出，为单片机系统供电。

## 二、参数计算

### 2.1 反馈电阻计算（Vout = 5V）

GD30DC1354 内部参考电压 Vref = 0.8V，输出电压公式：

```
Vout = Vref × (1 + R2/R3)
5V = 0.8V × (1 + R2/R3)
R2/R3 = 5/0.8 - 1 = 5.25
```

**实际选用：**
| 电阻 | 阻值 | 比值 | 计算Vout |
|------|------|------|---------|
| R2 (上分压) | 511kΩ | 511/105 = 4.867 | 0.8×5.867 = **4.69V** |
| R3 (下分压) | 105kΩ | — | — |

> **说明**：4.69V 是常见设计值（USB规范为5V±10%），若需精确5V，可按 R2/R3=5.25 调整，
> 例如 R2=499kΩ + R3=95.3kΩ。当前设计满足输出要求。

### 2.2 电感选型计算

```
L = (Vout × (Vin - Vout)) / (Vin × fsw × ΔIL × 0.3)
  = (5 × 19) / (24 × 1.2MHz × 0.5A × 0.3)
  ≈ 22μH
```

**实际选用**：15μH（FXL0630-150-M，长江微电）
- 开关频率 fsw ≈ 1.2MHz（GD30DC1354 内部设定）
- 电感饱和电流：3.5A（实际负载<1A ✅）

### 2.3 输入电容计算

```
Cin_min = Iout / (fsw × ΔVin)
        = 0.5A / (1.2MHz × 0.5V) ≈ 0.83μF
```

**实际选用：**
| 电容 | 作用 |
|------|------|
| C2 = 47μF 电解 | 输入储能、防跌落 |
| C1/C10/C11 = 100nF | 高频去耦（靠近芯片引脚） |

### 2.4 输出电容计算

```
Cout_min = ΔIload / (fsw × ΔVout)
         = 0.5A / (1.2MHz × 25mV) ≈ 16.7μF
```

**实际选用**：C5/C8 = 22μF × 2 = 44μF（村田GRT188R61A226ME13D）

### 2.5 自举电容

```
Cboot ≥ 10 × Cgate ≈ 47nF ~ 100nF
```

**实际选用**：C3 = 1μF（远大于最小要求，可靠自举）

### 2.6 补偿网络

R4（13.3kΩ）和 C6（10pF）构成 Type-II 补偿网络：
- 零点频率：fz = 1/(2π × R4 × C6) = 1/(2π × 13.3k × 10pF) ≈ 1.2MHz
- 确保环路稳定性

R6（100kΩ）为输出电压反馈上拉电阻。

## 三、主要元器件清单

| 位号 | 型号 | 作用 |
|------|------|------|
| U1 | GD30DC1354 | DC-DC降压转换芯片 |
| L1 | 15μH (FXL0630) | 储能电感 |
| Q1 | IRF9540NPBF | P-MOSFET（防反接保护） |
| D1 | MM1Z12 | 12V稳压二极管（保护） |
| SW2 | SS-12D10L5 | 电源开关 |
| LED1 | XL-2012UGC | 5V输出指示灯 |
| U4 | PZ254V-11-04P (4P排针) | 5V输出端子 |
| U2/U3/U5 | 2P排针 | 电流测量接口（短接帽） |

### 电流测量接口说明

在5V输出回路中串联了两个2P排针（U2、U3），正常工作用短接帽短接。
测量时断开短接帽，串入万用表电流挡测量输出电流。

## 四、PCB布局要点

1. 输入电容 C2（47μF）紧邻芯片引脚
2. 电感 L1 走线宽 ≥ 2mm，远离反馈回路
3. 反馈网络（R2/R3/R4/R6）远离电感和开关节点
4. 底层大面积GND覆铜
5. 大电流回路走线加宽
