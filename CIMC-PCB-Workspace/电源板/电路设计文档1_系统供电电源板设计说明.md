## 一、设计概述

本模块是基于GD30DC1354同步降压转换器构建的DC-DC电源电路，将外部输入的24V直流电压（支持18~36V宽范围输入）转换为稳定的5V直流输出，为整个系统提供电源供给。GD30DC1354是一款内置高低侧MOSFET的同步降压转换器芯片，开关频率为500kHz，内部参考电压$V_{ref}=0.6V$，最大输出电流可达3A。设计目标为赛题要求的5V±0.15V输出，额定负载2A。整个电路按照信号流向分为三个功能区域：输入保护与滤波电路、DC-DC降压转换电路、输出指示与接口电路。

## 二、输入保护与滤波电路

输入接口U5接入外部24V直流电源后，首先经过由Q1（IRF9540 P沟道MOSFET）构成的反接保护电路。IRF9540的源极接输入正极、漏极接后续电路、栅极通过电阻网络接地。当电源正常连接时，栅源电压$V_{GS}\approx -24V$，MOSFET导通为后续电路供电；当电源极性接反时，栅源电压$V_{GS}>0$，MOSFET截止，从而保护后级电路不受损坏。IRF9540的$V_{DSS}$额定值为-55V，远高于36V最高输入电压，具有充足的电压余量。

经过反接保护后，输入电压进入由C2（47μF/50V铝电解电容）与C3（1μF陶瓷电容）、C11（100nF陶瓷电容）构成的三级输入滤波网络。其中C2负责滤除低频纹波和输入电压波动，C3提供中频滤波和储能，C11紧邻芯片VIN引脚放置用于抑制高频开关噪声。该滤波网络为后续DC-DC转换器提供干净的输入电压，同时C2的50V耐压值相比36V最高输入留有38%的电压余量，确保长期工作可靠性。

## 三、DC-DC降压转换电路

### 3.1 主功率级

核心转换芯片U1（GD30DC1354）将输入电压通过内部高低侧MOSFET交替导通，驱动功率电感L1（15μH，型号FXL0630-150-M）完成降压转换。芯片的自举引脚（BST）通过C1（100nF自举电容）连接至开关节点SW，为高侧MOSFET驱动器提供高于输入电压的驱动电压。输出端由C4和C10（100nF高频陶瓷电容）与C5、C8（各22μF陶瓷电容）构成多级输出滤波网络，与L1配合将开关波形平滑为稳定的直流输出。芯片内部集成补偿网络，同时在FB引脚与输出之间并联C6（10pF）作为前馈电容跨接在R6两端（Vout至FB），用于提供高频相位提升以改善环路稳定性。

[图片：DC-DC转换电路主功率级原理图截图]

### 3.2 反馈分压网络与输出电压设定

输出电压由连接在输出端和FB引脚之间的电阻分压网络精确设定。上分压电阻R6（100kΩ）连接输出端至FB引脚，下分压电阻R4（13.3kΩ）连接FB引脚至GND。根据GD30DC1354数据手册的推荐值（Table 3），当R6=100kΩ、R4=13.3kΩ时，标称输出为5V，满足赛题5V±0.15V的要求。芯片内部参考电压Vref=0.6V（典型值），输出电压计算公式为：

$$V_{out} = V_{ref} \times (1 + \frac{R6}{R4})$$

选用±0.5%精度的贴片电阻R6和R4，在电阻精度最差组合下（R6=100.5kΩ, R4=13.233kΩ），输出电压的最大偏差为±0.045V（±0.9%），远优于赛题要求的±0.15V（±3%）容差，体现了高精度电阻选型带来的稳定输出优势。

[图片：反馈分压网络计算过程截图]

### 3.3 欠压锁定（UVLO）电路

芯片的使能引脚EN通过R2（511kΩ）上拉到输入电压、R3（105kΩ）下拉到GND构成UVLO分压网络。该网络设定系统的最小启动电压和停止电压，防止输入电压过低时芯片工作在不稳定状态。

[图片：UVLO分压电路原理图截图]

启动电压$V_{start}$和停止电压$V_{stop}$计算基于GD30DC1354数据手册的公式：

$$V_{start} = V_{EN\_RISING} \times \frac{R2+R3}{R3} + R2 \times (I_p + I_h)$$

其中$V_{EN\_RISING}=1.27V$为EN引脚上升阈值电压，$I_p=0.7\mu A$为EN引脚内部上拉电流，$I_h=2.1\mu A$为EN引脚迟滞电流。

$$V_{start} = 1.27 \times \frac{511k\Omega + 105k\Omega}{105k\Omega} + 511k\Omega \times (0.7\mu A + 2.1\mu A) = 1.27 \times 5.867 + 511k\Omega \times 2.8\mu A = 8.88V$$

停止电压$V_{stop}$计算公式为：

$$V_{stop} = V_{EN\_FALLING} \times \frac{R2+R3}{R3} + R2 \times I_p$$

其中$V_{EN\_FALLING}=1.23V$为EN引脚下降阈值电压。

$$V_{stop} = 1.23 \times 5.867 + 511k\Omega \times 0.7\mu A = 7.22 + 0.358 = 7.58V$$

UVLO启动电压8.88V和停止电压7.58V均远低于系统最低输入电压18V，确保系统在18~36V全输入范围内正常启动，且不会因输入电压的正常波动而意外关断。R2=511kΩ和R3=105kΩ的比值经过精心选配，使启动阈值和停止阈值之间有1.3V的迟滞窗口，有效防止了临界电压附近可能出现的反复开关振荡。

[图片：UVLO阈值计算推导过程截图]

### 3.4 电感选型计算

功率电感L1的取值直接影响输出纹波电流和系统效率。取输出额定电流$I_{out}=2A$，纹波电流系数$\Delta I_L = 25\% \times I_{out} = 0.5A$，开关频率$f_{sw}=500kHz$，在标称输入电压24V下电感值计算为：

$$L = \frac{(V_{in} - V_{out}) \times V_{out}}{V_{in} \times f_{sw} \times \Delta I_L}$$

代入数值：

$$L = \frac{(24V - 5V) \times 5V}{24V \times 500kHz \times 0.5A} = \frac{95}{6 \times 10^6} = 15.8\mu H$$

选取标准值15μH，型号为FXL0630-150-M（额定饱和电流3.5A）。在最低输入18V和最高输入36V下验证纹波电流和峰值电流：

在18V输入时，占空比$D=V_{out}/V_{in}=5/18=0.278$，纹波电流$\Delta I_L=(18-5)\times 0.278/(15\mu H\times 500kHz)=0.482A$，峰值电流$I_{pk}=2A+0.482/2=2.24A$。在24V输入时，占空比$D=0.208$，纹波电流0.500A，峰值电流2.25A。在36V输入时，占空比$D=0.139$，纹波电流0.575A，峰值电流2.29A。所有工况下峰值电流均低于L1的3.5A额定饱和电流，留有至少35%的电流余量，确保电感在全温度范围内不会进入饱和区。

[图片：电感选型计算与验证截图]

### 3.5 输出电容计算

输出端由C4（100nF高频陶瓷电容）、C5（22μF大容量陶瓷电容）、C8（22μF大容量陶瓷电容）、C10（100nF高频陶瓷电容）构成多级输出滤波网络，其中C5和C8并联总容值44μF，与电感L1配合抑制输出电压纹波。输出纹波电压近似计算为：

$$\Delta V_{out} = \frac{\Delta I_L}{8 \times f_{sw} \times C_{out}}$$

代入$\Delta I_L = 0.5A$、$f_{sw}=500kHz$、$C_{out}=44\mu F$（C5+C8并联）：

$$\Delta V_{out} = \frac{0.5A}{8 \times 500kHz \times 44\mu F} = 2.84mV$$

该纹波电压仅为输出电压的0.056%，远小于系统可接受范围。C5和C8选用10V耐压陶瓷电容，相对5V输出电压留有2倍电压余量；C4和C10选用50V耐压陶瓷电容，确保在启动浪涌和负载瞬态过程中不会因电压过冲而损坏。

[图片：输出纹波计算截图]

## 四、输出指示与接口电路

输出电压经滤波后连接到输出接口U6（4P排针，引脚配置为5V、GND、5V、GND），双路输出的设计为后续功能板卡提供了独立的供电通路，便于各板卡独立供电和电流测量。同时通过限流电阻R5（620Ω）驱动LED1作为电源指示灯，LED选用绿色发光二极管，正向压降约2V，工作电流计算为：

$$I_{LED} = \frac{V_{out} - V_F}{R5} = \frac{5V - 2V}{620\Omega} \approx 4.8mA$$

4.8mA的驱动电流在保证LED清晰可见亮度（通常在2~10mA范围内）的同时，将LED指示电路的功耗控制在约24mW，对系统整体效率的影响可忽略不计。

[图片：输出接口与LED指示电路截图]

电流测量接口由U2和U3（2P排针+短接帽）构成，分别串接在两组输出回路中。正常工作时短接帽保持短路状态使电流直接通过；需要测量输出电流时取下短接帽，在两端串入万用表的电流档位即实现将万用表串联进入输出回路进行测量。该方案相比在PCB上预留焊盘测量点的方式更为便捷，无需焊接即可完成现场测试。

## 五、BOM清单

核心器件U1选用GD30DC1354同步降压转换器芯片，该芯片内置高低侧MOSFET，外围元件需求少。功率电感L1选用FXL0630-150-M型15μH电感，额定饱和电流3.5A满足2A负载需求。输入保护采用IRF9540 P沟道MOSFET实现防反接，$V_{DSS}=-55V$留有余量。输入滤波电容由C2（47μF/50V铝电解电容）、C3（1μF陶瓷电容）、C11（100nF陶瓷电容）共同构成三级滤波网络。自举电容C1选用100nF陶瓷电容，为芯片高侧MOSFET驱动器供电。输出滤波电容由C4（100nF）、C5（22μF/10V）、C8（22μF/10V）、C10（100nF）构成多级输出滤波网络。反馈分压网络电阻R6=100kΩ和R4=13.3kΩ选用±0.5%精度电阻以保证输出精度，使全温度范围输出电压波动控制在±0.9%以内。UVLO设置电阻R2=511kΩ和R3=105kΩ选用±1%精度电阻即可满足8.88V启动阈值和7.58V停止阈值的设定要求。LED限流电阻R5为620Ω，补偿电容C6=10pF跨接在R6两端构成前馈补偿。接口方面，U4为4P排针提供5V双路输出，U5为2P接线端子接入24V输入，U2和U3为2P排针配短接帽实现电流测量。全部阻容元件均采用0603封装，在满足功率需求的同时保持较小的PCB面积占用。

## 六、宽电压输入分析

本设计需在18~36V宽输入电压范围内稳定工作。在不同输入电压下，占空比$D=V_{out}/V_{in}$随输入电压升高从0.278线性下降到0.139，均在GD30DC1354芯片的占空比可控范围内。导通时间$T_{on}=D/f_{sw}$在36V高输入下为278ns，远高于GD30DC1354最小导通时间100ns的限制，确保不会出现脉冲跳跃导致的输出不稳问题。功率电感L1在各种输入下的峰值电流均控制在3.5A额定值以内，最大峰值2.29A留有余量35%。输入电容C2的50V耐压相对36V最高输入留有38%电压余量，P-MOSFET Q1的$V_{DSS}$耐压-55V相对-36V留有34%电压余量。芯片U1的VIN引脚绝对最大额定值为36V，与系统最高输入电压一致，在实际应用中留有充足的安全边界。整个电源板从输入反接保护、三级滤波网络到输出多级滤波的设计层层把关，各关键器件耐压等级均高于实际工作电压30%以上，确保系统在18~36V全输入范围内长期稳定可靠运行。
