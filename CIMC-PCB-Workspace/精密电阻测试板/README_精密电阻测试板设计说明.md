# 精密电阻模拟测试板设计说明

## 一、设计目标

板载12颗精密电阻，通过拨码开关选择单电阻接入采样板，模拟PT100在不同温度下的阻值。按三线制PT100引脚设计(2红1白)，用于对采样板进行功能验证和精度标定。

## 二、关于"缺153Ω"的说明

实际这12个电阻覆盖了：

|------|------|------|---------|------|
| **基础组 A (DIP-6A)** | | | | |

| SWA2 | RA2 | 107Ω | **≈17.8°C** | 赛题要求20°C(107.79Ω)最接近E96值 |
| SWA3 | RA3 | 115Ω | **≈38.6°C** | 赛题要求40°C(115.54Ω)最接近E96值 |

## 三、阻值选择逻辑

### 3.1 基础组（对应赛题6个温度点）

赛题标准阻值在E96（±0.1%）标准系列中的匹配：

| 理论值 | E96标称值 | 偏差 | 反算温度 | 可接受? |
|-------|----------|------|---------|--------|
| 100.00Ω | 100Ω | 0% | 0.0°C | ✅ |
| 107.79Ω | 107Ω | -0.73% | 17.8°C | ✅赛题允许 |
| 115.54Ω | 115Ω | -0.47% | 38.6°C | ✅赛题允许 |
| 123.24Ω | 124Ω | +0.62% | 62.3°C | ✅赛题允许 |
| 130.90Ω | 130Ω | -0.69% | 77.9°C | ✅赛题允许 |
| 138.51Ω | 140Ω | +1.08% | 103.9°C | ✅赛题允许 |

> 赛题明确允许"选择最接近的标准阻值，重新换算理论温度，评分以实际阻值为准"

### 3.2 进阶组（扩展温区验证）

80.6Ω和82.5Ω**本身就是E96标准值**（E96系数表包含8.06和8.25），可直接采购：
- 80.6Ω = 8.06 × 10¹ ✅
- 82.5Ω = 8.25 × 10¹ ✅

113/115/150/154也是E96标准值。

## 四、切换机构设计

### 4.1 双DIP-6方案

使用两个独立DIP-6拨码开关（DS-06RP），每个6路SPST：

- DIP-6(A)：控制基础组6个电阻
- DIP-6(B)：控制进阶组6个电阻

**引脚映射**（以DS-06RP为例）：
```
DIP-6 俯视图：
        ┌─────────────────────┐
   SW1  │  1▉ 2              │ → 1=COM, 2=OUT
   SW2  │  3▉ 4              │ → 3=COM, 4=OUT
   SW3  │  5▉ 6              │ → 5=COM, 6=OUT
        │                     │
   SW4  │  7▉ 8              │ → 7=COM, 8=OUT
   SW5  │  9▉ 10             │ → 9=COM, 10=OUT
   SW6  │  11▉ 12            │ → 11=COM, 12=OUT
        └─────────────────────┘
```

> 注：各品牌引脚定义可能不同，焊接前用万用表确认COM和OUT引脚

### 4.2 为什么不能用DIP-12

- DIP-12不常见（最大DIP-8），难买
- 两个DIP-6对称布局更整齐
- 物理分隔基础组/进阶组，防误操作

### 4.3 防误操作安全规则

⚠️ **12个开关同一时刻只能有1个ON！否则两电阻并联，读数完全错误**

误操作后果举例：100Ω∥80.6Ω = 44.6Ω（对应约-145°C，无意义）

## 五、输出接口设计

### 5.1 J1排母引脚定义

与采样板上U6排针对插：

```
测试板 J1(母)      采样板 U6(公)
 J1.1 SENSE-  ═══  U6.1 → INA333 -IN
 J1.2 SENSE+  ═══  U6.2 → 恒流源输出(1mA)
 J1.3 RTN     ═══  U6.3 → R2.2 → AGND
```

### 5.2 电流路径

```
1mA恒流 → U6.2 → J1.2(COM总线) → 被选中的DIP开关
  → 精密电阻Rx → 冷端母线(BUS_GND) → J1.1(SENSE-) + J1.3(RTN)
  → U6.1(INA333 -IN) + U6.3(AGND)

INA333测得：Vdiff = V(J1.2) - V(J1.1) = 1mA × Rx
```

## 六、ADC量程验证

GD30AD3344差分FSR=±2.048V（以2.5V为共模基准），所有12个挡位验证：

| 挡位 | 阻值 | Vdiff(×1mA) | INA333输出 | 占FSR | 判断 |
|------|------|-------------|-----------|-------|------|
| SWB1 | 80.6Ω | 80.6mV | 2.5V+0.806×22.05=4.27V→-0.723V | 32.4% | ✅ |
| SWB2 | 82.5Ω | 82.5mV | 2.5V+0.825×22.05=4.32V→-0.681V | 33.4% | ✅ |
| ... | ... | ... | ... | ... | ✅ |
| SWB6 | 154Ω | 154mV | 2.5V+1.54×22.05=5.90V→+0.896V | 71.9% | ✅ |

全部落在 **32.4%~71.9% FSR**（赛题要求20%~80%），直接插上用即可 ✅

### 7.3 关键布线规则

**① BUS_GND（最重要）**
- 所有12个电阻冷端 + J1.1 + J1.3 连接成同一网络
- 用大面积覆铜处理，不要用细走线串接
- 走线宽度 ≥ 1.5mm，推荐2mm覆铜
- 总阻抗目标 < 10mΩ

**② BUS_COM**
- 两个DIP-6所有COM端 + J1.2 连接
- 走线宽度 ≥ 1mm
- 走顶层，绕开电阻区域

**③ N_A1~N_A6 / N_B1~N_B6**
- DIP-6的OUT脚到对应电阻的走线
- 尽可能短，宽度 ≥ 0.5mm
- 不要绕大圈

### 7.4 覆铜方案

| 层 | 覆铜网络 | 区域 |
|----|---------|------|
| 顶层 | BUS_GND | 板子中部到下部的电阻区域+冷端 |
| 底层 | 不铺(或GND) | 可选 |

- 覆铜间距：0.3mm
- 过孔：6~8个（0.5mm孔/0.9mm盘），连接顶层覆铜
