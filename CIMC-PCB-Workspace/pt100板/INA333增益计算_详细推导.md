# INA333增益计算详细推导（修正版）

> **致谢**：感谢用户指出FSR范围计算和INA333拓扑误解，以下为修正后版本。

## 一、电路拓扑确认

### 实际连接（从网表提取）

```
恒流源(1mA) ——— 红线(激励) ——— PT100 ——— GND
                                            ↑
红线(补偿) ——— INA333 +IN(U3.3)
GND ——— INA333 -IN(U3.2)
GND ——— INA333 REF(U3.5)

INA333输出(U3.6) ——— GD30AD3344 AIN0(U4.4)
LM4040A25 2.5V ———— GD30AD3344 AIN3(U4.7) [作为差分负端]
```

**关键发现**：INA333的REF引脚接GND，非2.5V！
- Vout_333 = REF + (Vin+ - Vin-) × G = 0 + V_PT100 × G
- **Vout_333 = V_PT100 × G**（单端输出，相对于GND）

### ADC差分输入
```
Vdiff = V(AIN0) - V(AIN3) = Vout_333 - 2.5V
```

## 二、关键参数

### GD30AD3344配置
- PGA = 010 → FSR = **±2.048V**
- 总范围 4.096V（从 -2.048V 到 +2.048V）
- 分辨率：24位（有效分辨率取决于OSR设置）

### PT100在1mA激励下的压降

| 温度 | PT100电阻 | 压降V_PT100 |
|------|----------|------------|
| -50°C | 80.31Ω | 80.31mV |
| 0°C | 100.00Ω | 100.00mV |
| +50°C | 119.40Ω | 119.40mV |
| +100°C | 138.51Ω | 138.51mV |
| +150°C | 157.33Ω | 157.33mV |

### INA333增益公式
```
G = 1 + 100kΩ / Rg
其中Rg为外部增益设置电阻（位号R4）
```

## 三、赛题要求：信号占ADC满量程20%~80%

### FSR范围理解（关键更正）

FSR = ±2.048V，总范围 4.096V

```
-2.048V ──── -1.229V ──── 0V ──── +1.229V ──── +2.048V
  0%          20%        50%       80%           100%
```

- **20%位置** = -2.048V + 4.096V × 20% = -2.048V + 0.8192V = **-1.229V**
- **80%位置** = -2.048V + 4.096V × 80% = -2.048V + 3.2768V = **+1.229V**

信号Vdiff必须满足：**-1.229V ≤ Vdiff ≤ +1.229V**

## 四、增益选择：R4=4.75kΩ → G=22.05

```
G = 1 + 100kΩ / 4.75kΩ = 1 + 21.05 = 22.05
```

### 约束条件推导

**约束1（-50°C不低于20%）：**
```
Vdiff(-50°C) = 22.05 × 80.31mV - 2.5V = 1.771V - 2.5V = -0.729V
-0.729V ≥ -1.229V ✅ （余量0.500V）
```

**约束2（+150°C不超80%）：**
```
Vdiff(+150°C) = 22.05 × 157.33mV - 2.5V = 3.469V - 2.5V = +0.969V
+0.969V ≤ +1.229V ✅ （余量0.260V）
```

**增益可行范围：**
```
约束1（-50°C）：G × 80.31mV - 2.5V ≥ -1.229V → G ≥ 15.83
约束2（+150°C）：G × 157.33mV - 2.5V ≤ +1.229V → G ≤ 23.70
```
**15.83 ≤ G ≤ 23.70** → G=22.05在范围内 ✅
