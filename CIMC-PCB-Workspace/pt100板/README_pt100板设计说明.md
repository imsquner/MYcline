# PT100温度采样板设计说明

## 一、设计目标

将PT100铂电阻温度传感器（-50°C~+150°C）的电阻变化转换为电压信号，经GD30AD3344 ADC数字化后通过SPI接口输出。

## 二、信号链架构

```
PT100/测试板 → 恒流源(1mA) → 差分电压 → INA333(×22.05) → RC滤波 → GD30AD3344(24bit ADC) → SPI → MCU
                                     ↑
                              2.5V 基准电压源 (LM4040A25)
```

## 三、参数详细计算

### 3.1 恒流源电路（1mA激励）

**方案**：基于MCP6004运放+MMBT3904晶体管+VREF构成的Howland电流源变形

**原理**：
- LM4040A25提供2.5V基准电压
- R1 = 2.5kΩ，通过MCP6004缓冲后驱动MMBT3904基极
- 发射极电流 Ie = 2.5V / 2.5kΩ = 1mA ✅

```
Ic ≈ Ie = 1mA（β >> 100，基极电流可忽略）
```

**设计选择理由**：
- 1mA激励电流：PT100自热功率仅P = I²R = 1mA²×100Ω = 0.1mW，自热误差<0.01°C ✅
- LM4040A25精度±0.1%（初始）+ 20ppm/°C温漂，适用于工业测温
- MCP6004为轨到轨运放，RRIO特性，3.3V供电下可输出0~3.3V

### 3.2 三线制PT100接线原理

```
           ┌─────────┐
           │   恒流源 │──── I=1mA ──── 红线(激励) ────┐
           └─────────┘                                │
                                                      ├── PT100 ── 白线 ──→ GND
           ┌─────────┐                                │
           │ INA333  │◄── 红线(补偿) ─────────────────┘
           │ +IN ────┤
           │ -IN ────┤
           └─────────┘
```

红线(补偿)提供引线电阻补偿：两条等长红线电阻相同，INA333差分测量自动抵消引线压降。

### 3.3 放大倍数设计

**PT100在-50°C~+150°C的阻值范围**（IEC 60751标准，α=0.00385055）：

| 温度 | 电阻值 | 恒流源压降 |
|------|--------|-----------|
| -50°C | 80.31Ω | 80.31mV |
| 0°C | 100.00Ω | 100.00mV |
| 50°C | 119.40Ω | 119.40mV |
| 100°C | 138.51Ω | 138.51mV |
| 150°C | 157.33Ω | 157.33mV |

**输入差分电压范围**：80.31mV ~ 157.33mV（ΔV ≈ 77mV）

**GD30AD3344 ADC量程**：差分模式 FSR = ±2.048V（AIN0 - AIN3）
- ADC共模偏置 = Vref/2 = 2.5V/2 = 1.25V（不对，看赛题安排）
- 实际共模基准由外部2.5V提供

**INA333增益计算**：

赛题要求：-50°C~+150°C对应信号在ADC满量程的20%~80%范围内。

目标Vout范围：2.5V ± 2.048V = 0.452V ~ 4.548V
实际使用20%~80%：0.452 + 2.048×20% ~ 0.452 + 2.048×80%
即 0.862V ~ 4.138V（以2.5V为中心）

但更常用的设计：
- 让0°C（100Ω, 100mV）的输出落在2.5V附近
- Vout(0°C) = 2.5V + 100mV × G
- 若Vout(0°C) ≈ 2.5V（即共模基准），则G=(Vout-2.5V)/Vin

**实际增益计算：**

以100°C为例：138.51mV，希望Vdiff贴近ADC量程50%左右
- 目标：Vout - 2.5V ≈ ±2.048V × 50% = 1.024V
- G = (Vout - 2.5V) / (R_pt100 × 1mA)
- 选中间温度50°C（119.4mV）让Vout≈2.5V+0.5V=3.0V→G=0.5/0.1194≈4.19

**实际选用的增益设置电阻**：
INA333增益公式：G = 1 + (100kΩ / Rg)
- 选用 R5 = Rg = 4.75kΩ
- G = 1 + 100/4.75 = 1 + 21.05 = **22.05 ✅**

**各温度点验证（以2.5V为共模基准）：**

| 温度 | 电阻 | Vin | Vout = 2.5 + Vin×22.05 | Vdiff | 占FSR |
|------|------|-----|----------------------|-------|-------|
| -50°C | 80.31Ω | 80.31mV | 2.5+1.771=4.271V | +1.771V | 43.2% |
| 0°C | 100.00Ω | 100.00mV | 2.5+2.205=4.705V | +2.205V | 53.8% |
| 50°C | 119.40Ω | 119.40mV | 2.5+2.633=5.133V | +2.633V | 64.3% |
| 100°C | 138.51Ω | 138.51mV | 2.5+3.054=5.554V | +3.054V | 74.6% |
| 150°C | 157.33Ω | 157.33mV | 2.5+3.469=5.969V | +3.469V | 84.7% |

> **结论**：Vdiff范围 43.2%~84.7%，覆盖赛题要求的20%~80% ✅
> ADC电压最高5.969V（3.3V供电下需注意输入范围，GD30AD3344允许Vdd+0.3V）

### 3.4 低通滤波设计

**RC低通滤波器**（R7=1.5kΩ, C6=220nF）：
- 截止频率：fc = 1/(2π×1.5k×220nF) = 1/(2π×330μ) ≈ **482Hz**
- 衰减50Hz工频干扰：Atten(50Hz) ≈ -40dB/dec × log(fc/50) ≈ 非常强
- 有效抑制工频干扰 ✅

**ADC输入端滤波**（R8/R9/R10/R11=50Ω，C7/C9=100nF/10μF）：
- 多级RC滤波，截止频率约32kHz
- 在ADC输入端提供抗混叠保护

### 3.5 基准电压源

**LM4040A25IDBZR**：
- Vref = 2.5V ± 0.1%
- 温漂 20ppm/°C（典型）
- SOT-23封装，用于INA333共模偏置和ADC参考

**去耦电容**：C1(100nF)紧邻U1引脚，确保低噪声

### 3.6 ADC配置说明

**GD30AD3344**（MSOP-10）：
- 分辨率：24位（Σ-Δ ADC）
- 满量程：±2.048V（差分模式）
- SPI接口：SCLK/MOSI/MISO/CS
- 转换速率可编程

**外部参考**：AIN3作为外部参考输入，连接2.5V

**差分输入**：
- AIN0(+) → INA333输出
- AIN3(-) → 2.5V基准（作为差分负端）

| U4 | GD30AD3344 | 24位Σ-Δ ADC |

## 五、PCB布局要点

1. **模拟/数字分区**：INA333和ADC区域为模拟区，SPI接口为数字区
2. **单点接地**：模拟地和数字地通过0Ω电阻(R12)单点连接
3. **基准源去耦**：LM4040A25输出电容紧邻引脚
4. **差分走线**：INA333输入走线等长等宽，避免环路面
5. **电源去耦**：每对VDD/GND引脚旁放置100nF电容
