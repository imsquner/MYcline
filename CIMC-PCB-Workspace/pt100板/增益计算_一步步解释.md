# PT100增益计算：一步步拆解说明

> 回应你对"负电压"的疑问，从最基础的原理说起

## 第1步：电路在做什么？

```
恒流源(1mA) → PT100(100Ω@0°C) → 产生100mV电压
                                              ↓
INA333放大22.05倍 → Vout=100mV×22.05=2.205V
                                              ↓
送进ADC的AIN0引脚
```

同时，ADC的另一个输入引脚AIN3接了**2.5V基准电压**。

**ADC做的是减法：** 读取 `AIN0 - AIN3` 的差值

所以：
```
ADC读数 = Vout(INA333) - 2.5V
```

## 第2步：为什么会出现负数？

因为INA333的输出电压在不同温度下是：

| 温度 | PT100压降 | INA333输出(Vout) | ADC测量值(Vout - 2.5V) |
|------|----------|-----------------|----------------------|
| -50°C | 80.31mV | **1.771V** | **1.771 - 2.5 = -0.729V** |
| 0°C | 100.00mV | **2.205V** | **2.205 - 2.5 = -0.295V** |
| +50°C | 119.40mV | **2.633V** | **2.633 - 2.5 = +0.133V** |
| +100°C | 138.51mV | **3.054V** | **3.054 - 2.5 = +0.554V** |
| +150°C | 157.33mV | **3.469V** | **3.469 - 2.5 = +0.969V** |

**关键原因**：INA333输出从1.771V到3.469V，2.5V在中间。所以：
- 低温时(<10°C)：Vout < 2.5V → ADC读到**负值**
- 高温时(>10°C)：Vout > 2.5V → ADC读到**正值**

这是差分测量的正常现象，GD30AD3344作为差分ADC天然支持正负输入。

## 第3步：你说的对——应该换个角度理解"量程"

你质疑"怎么还有负电压"，其实是在说：赛题要求的"ADC满量程20%~80%"不应该包含负电压。

让我用两种方式解释：

### 方式A：按差分全范围 ±2.048V 来算（我之前用的）

```
满量程范围: -2.048V 到 +2.048V (总跨度4.096V)
20%位置 = -2.048 + 4.096×20% = -1.229V
80%位置 = -2.048 + 4.096×80% = +1.229V
```

检查Vdiff范围 -0.729V ~ +0.969V：
- -0.729V > -1.229V ✅ 超过20%下限
- +0.969V < +1.229V ✅ 未超80%上限

### 方式B：按单端0~+2.048V来算（你更认同的方式）

```
满量程范围: 0V 到 +2.048V (总跨度2.048V)
20%位置 = 0 + 2.048×20% = 0.410V
80%位置 = 0 + 2.048×80% = 1.638V
```

但Vdiff有负值（-0.729V, -0.295V），不满足≥0.410V的要求。

## 第4步：问题出在哪？

**根本问题在于信号链的参考点选择。**

目前的设计是：
```
ADC差分参考点 = 2.5V（LM4040A25基准）
```
所以信号是围绕2.5V上下摆动的。

如果要让ADC输入始终为正，有两个改法：

### 改法1：把INA333的REF引脚接到2.5V
```
原设计: INA333 REF接GND → Vout = V_PT100 × G
改进:   INA333 REF接2.5V → Vout = 2.5V + V_PT100 × G
```

这样Vout范围变为 2.5+1.771=4.271V 到 2.5+3.469=5.969V，Vdiff全为正。

但问题是：**5.969V超过5V供电轨了**（需要降低增益或改用更高供电电压）。

### 改法2：把ADC基准从2.5V改为更低的电压
```
例如将AIN3接1.5V基准
Vdiff(-50°C) = 1.771 - 1.5 = +0.271V（为正了）
```
但这需要额外增加一个基准芯片。

## 第5步：所以你的意思是什么？

你能不能告诉我：**赛题对"ADC满量程20%~80%"的具体定义是什么？**

1. ❓ 按ADC正半轴（0~+2.048V）算，信号必须全部为正且在20%~80%内？
2. ❓ 按ADC差分全范围（±2.048V）算，允许信号为负但在范围内？

这个定义不同，计算方法和器件选型都会完全不同，需要你确认。
