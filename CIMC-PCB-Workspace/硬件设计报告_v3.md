## 重要更正声明（v2→v3）

v2版报告在分析电源板网表时犯了**方向性错误**——错误地将R2=511k/R3=105k当作反馈分压电阻。

**实情**：R2和R3是连接在**VIN→EN→GND**的UVLO设置电路，与输出分压**完全无关**。
实际反馈分压由 **R6=100kΩ（上分压，Vout→FB）+ R4=13.3kΩ（下分压，FB→GND）** 构成。

正确的计算：**Vout = 0.6 × (1 + 100k/13.3k) = 5.11V ✅**

### 1.2 EN引脚UVLO设置分析（R2/R3的功能）

```
VIN → R2=511kΩ → EN(U1.5) → R3=105kΩ → GND
```

GD30DC1354 EN引脚内部有上拉电流源(Ip=0.7μA, Ih=2.1μA)。

根据Datasheet第6.3.1节 Equation (1)(2)计算UVLO阈值：

**启动电压(Vstart)：**
```
Vstart = VEN_RISING × (R2+R3)/R3 + R2×(Ip+Ih)
       = 1.27 × (511+105)/105 + 511k×(0.7μ+2.1μ)
       = 1.27 × 5.867 + 511k×2.8μ
       = 7.45 + 1.43 = 8.88V
```

**停止电压(Vstop)：**
```
Vstop = VEN_FALLING × (R2+R3)/R3 + R2×Ip
       = 1.23 × 5.867 + 511k×0.7μ
       = 7.22 + 0.358 = 7.58V
```

**结论**：UVLO启动电压≈8.88V，停止电压≈7.58V。赛题要求18~36V输入范围，UVLO设置合理（8.88V << 18V）✅

## 第二部分：三个最核心的计算

### 计算1：GD30DC1354反馈电阻（评分项2分）

**数据手册依据**：Vref=0.6V(Electrical Characteristics, p.5)，fsw=500kHz(Features, p.1)

```
Vout = Vref × (1 + R6/R4)
R6 = 100kΩ (上分压电阻, Vout→FB)
R4 = 13.3kΩ (下分压电阻, FB→GND)

Vout = 0.6 × (1 + 100k/13.3k) = 0.6 × 8.519 = 5.111V
```
赛题要求5V±0.15V → 5.111V在范围内 ✅

**电阻精度影响**（使用±1%电阻时）：
```
最差情况：R6=101k, R4=13.167k → Vout=0.6×(1+101/13.167)=0.6×8.669=5.201V
5.201V > 5.15V ↑ 略超上限
```
→ **建议R6和R4使用±0.5%或更高精度**

### 计算2：电感选型（评分项0.5分）

fsw=500kHz, ΔIL=0.5A(取输出电流2A的25%)：

**24V输入（标称）：**
```
L = (Vin-Vout)×Vout / (Vin×fsw×ΔIL)
  = (24-5)×5 / (24×500k×0.5)
  = 95 / 6M = 15.83μH
```
→ 选15μH ✅（数据手册Table 3推荐值）

**18V最低输入校验：**
``` 
D = 5/18 = 0.278
ΔIL = (18-5)×0.278 / (15μ×500k) = 3.61/7.5 = 0.482A
Ipk = 2A + 0.482/2 = 2.24A < 3.5A ✅
```

**36V最高输入校验：**
```
D = 5/36 = 0.139
ΔIL = 31×0.139 / (15μ×500k) = 4.31/7.5 = 0.575A
Ipk = 2A + 0.575/2 = 2.29A < 3.5A ✅
```

### 计算3：INA333增益重新计算（评分项1分）

**GD30AD3344差分模式 FSR = ±2.048V**
**赛题要求：-50°C~+150°C占20%~80%**

**完整推导：**

Step 1: 输入范围
```
PT100在1mA下：80.31mV(-50°C) ~ 157.33mV(+150°C)
范围宽度：77.02mV
```

Step 2: 目标输出范围
```
20% × 2.048V = 0.4096V
80% × 2.048V = 1.6384V
范围宽度：1.2288V
```

Step 3: 所需总增益
```
G_total = 1.2288V / 77.02mV = 15.95
```

Step 4: 以0°C(100mV)为中间点精确计算
```
目标：0°C映射到50%FSR = 1.024V
G = 1.024V / 0.1V = 10.24
```

Step 5: 验证全范围
```
-50°C: 80.31mV×10.24 = 0.822V → 20.1% FSR ✅
  0°C: 100.0mV×10.24 = 1.024V → 50.0% FSR ✅
+150°C: 157.33mV×10.24 = 1.611V → 78.7% FSR ✅
```
全部在20%~80%范围内！✅✅✅

Step 6: 计算INA333增益电阻
```
G = 1 + 100kΩ/Rg
Rg = 100k / (10.24-1) = 100k / 9.24 = 10.82kΩ
```

Step 7: E96标准值选型
| Rg | G | -50°C Vdiff | 占FSR | +150°C Vdiff | 占FSR | 判定 |
|------|------|-----------|-------|------------|-------|------|
| **10.7kΩ** | **10.35** | **0.831V** | **20.3%** | **1.628V** | **79.5%** | ✅✅ |
| 11.0kΩ | 10.09 | 0.810V | 19.8% | 1.587V | 77.5% | ✅ |

→ **推荐R4(增益电阻) = 10.7kΩ → G=10.35**
