#### ▶ 关键问题1：Vout实测将仅为3.52V（严重不达标）

**根本原因**：反馈电阻组合(R2=511kΩ, R3=105kΩ)对应Vref=0.8V设计，但芯片实际Vref=0.6V。

**正确计算**（GD30DC1354 Datasheet Equation (3)，第11页）：
```
Vout = Vref × (1 + R2/R3)
其中：Vref = 0.6V，R2为上分压电阻(连接到Vout)，R3为下分压电阻(连接到GND)

当前值：Vout = 0.6 × (1 + 511k/105k) = 0.6 × 5.867 = 3.52V ❌
```

**3.52V远低于赛题要求的5V±0.15V，必须修改！**

**修正方案**——直接采用数据手册推荐值（Table 3, 第11页）：

| 方案 | R2（上分压） | R3（下分压） | Vout | 判定 |
|------|------------|------------|------|------|
| **数据手册推荐** | **100kΩ** | **13.3kΩ** | **0.6×(1+100/13.3)=5.11V** | ✅±0.15V |
| 备选1 | 105kΩ | 14.0kΩ | 0.6×(1+105/14)=5.10V | ✅ |
| 备选2 | 110kΩ | 14.7kΩ | 0.6×(1+110/14.7)=5.09V | ✅ |

**建议**：保留现有R6=100kΩ（原标为R6=100kΩ，看BOM中标为R6=100kΩ），将R3从105kΩ换为**13.3kΩ**，匹配数据手册推荐方案：
```
Vout = 0.6 × (1 + 100k/13.3k) = 0.6 × 8.519 = 5.11V ✅
```
并且R6(R2)和R4(R4)的位号不变，仅需重买R3=13.3kΩ。

**但是注意**：网表中R6=100kΩ连接的是L1.2和C6.1...等等，让我重新核对：

网表中关于R6的是：'$2N3' ; C6.1 L1.2 R6.2 U2.2 
而且数据手册推荐电路图(Figure 4, 第10页)中：
- R1 = 100kΩ(上分压) - 对应我们的R2=511kΩ
- R2 = 13.3kΩ(下分压) - 对应我们的R3=105kΩ

且BOM中R6=100kΩ是补偿网络的一部分：
'$2N5' ; C6.2 R4.2 R6.1 U1.4 → 这连的是FB引脚

查看典型电路图（Figure 4）：
VOUT → R1(100k) → FB → R2(13.3k) → GND
                  FB → R4(13.3k) + C5(75pF) → 内部补偿

所以我们的BOM中R4=13.3kΩ + C6=10pF 其实是补偿网络（数据手册中C5=75pF），但GD30DC1354的datasheet第1页Features写明"Internal Compensation Function"——内部补偿！

第6.2.3节说："The frequency compensation components are placed internal between the output of the error amplifier and ground."

所以**外部R4/C6是可选的**。但数据手册Figure 4还是画了R4(13.3k) + C5(75pF)的推荐电路。保留没问题。

**综合修正方案**：
- 将 **R2=511kΩ 改为 100kΩ**（上分压电阻）
- 将 **R3=105kΩ 改为 13.3kΩ**（下分压电阻）
- 保留R4=13.3kΩ（补偿网络，与数据手册一致）
- Vout = 0.6 × (1 + 100/13.3) = **5.11V** ✅

#### ▶ 关键问题2：恒流源电路命名混乱

网表/BOM分析（基于BOM SCH_pt100）：
- **R1=10kΩ** → 连接5V和运放U2.12(电源脚)，实际是**运放电源去耦/上拉**，不是电流设置电阻
- **R2=2.5kΩ** → 连接Q1.2基极路径，实际是**电流设置电阻**
- 之前README文档把R1误称为电流设置电阻

**正确的恒流源电路解读**：
```
LM4040A25(U1)输出2.5V基准
    ↓
MCP6004(U2)运放缓冲（同相跟随）
    ↓ 运放输出→Q1基极
R2=2.5kΩ（接在Q1发射极到地）
    ↓
Ie = 2.5V / 2.5kΩ = 1mA
    ↓
Q1集电极输出 ≈ 1mA → U6.2 (PT100激励端)
```

**建议**：在原理图中将R2的位号改为**R_SET**或**R_CURRENT**以消除歧义；文档中明确R1=10kΩ是运放供电无关电阻。

#### ▶ 关键问题3：三线制PT100接线可靠性验证

网表关键节点：
```
$1N15 : Q1.3(恒流输出) + U3.3(INA333 +IN) + U6.2(PT100接口脚2)
$1N17 : U3.2(INA333 -IN) + U6.1(PT100接口脚1)
$1N7  : Q1.2(发射极) + R2.2 + U2.13(运放输出) + U6.3(PT100接口脚3)
```

**三线制标准接法解读**：
```
J1脚2(SENSE+) ← 红线(激励) ← PT100一端
J1脚1(SENSE-) ← 红线(补偿) ← PT100另一端(同点)
J1脚3(RTN)    ← 白线(返回) ← PT100另一端(同点)
```

恒流路径：Q1集电极 → U6.2 → 红线(激励) → PT100 → 红线(补偿) → U6.1 → INA333 -IN
返回路径：PT100 → 白线(返回) → U6.3 → R2.2 → GND

INA333差分：+IN(U3.3)接到U6.2端，-IN(U3.2)接到U6.1端
→ 测得 Vdiff = V(U6.2) - V(U6.1) = 1mA × R_PT100 ✅

**接线逻辑正确**。但实际连接中，PT100的"红线(补偿)"和"红线(激励)"在外部是PT100内部的同一个节点引出的两个等长红线，差分放大器和恒流源通过不同的红线连接PT100两端——当前电路正确实现了这一功能。

#### ▶ 关键问题4：INA333增益需重新优化

ADC差分输入FSR = ±2.048V。要求-50°C~+150°C占20%~80%：
- 目标范围：0.4096V ~ 1.6384V
- PT100电压：80.31mV ~ 157.33mV

让0°C(100mV)在50%(1.024V)：
```
G = 1.024 / 0.1 = 10.24
```

校验：
```
-50°C: 80.31mV × 10.24 = 0.822V → 20.1% ✅ 接近20%
+150°C: 157.33mV × 10.24 = 1.611V → 78.7% ✅ 接近80%
```

INA333 增益公式 G = 1 + 100kΩ/Rg：
```
Rg = 100k / (10.24 - 1) = 100k / 9.24 = 10.82kΩ
```

E96标准值选型：
- 10.7kΩ → G = 1 + 100/10.7 = 10.35 → 最大Vdiff = 1.628V → 79.5% ✅
- 11.0kΩ → G = 1 + 100/11.0 = 10.09 → 最大Vdiff = 1.587V → 77.5% ✅

**建议**：将R4(增益设置电阻)从 **4.75kΩ 改为 10.7kΩ**，使G=10.35。

## 第二部分：核心参数详细计算

### 2.1 GD30DC1354反馈电阻计算（完整推导）

**依据**：Datasheet 第7.3.1节 Equation (3)
```
Vout = Vref × (1 + R2/R3)
Vref = 0.6V（典型值，Datasheet第5页）
```

**目标**：Vout = 5.0V

**Step 1：计算分压比**
```
R2/R3 = Vout/Vref - 1 = 5.0/0.6 - 1 = 7.333
```

**Step 2：选择标准电阻值**
数据手册Table 3（第11页）直接给出5V输出的推荐值：
- R2 = **100kΩ**（E96标准值）
- R3 = **13.3kΩ**（E96标准值）

**Step 3：验证**
```
Vout = 0.6 × (1 + 100k/13.3k) = 0.6 × 8.519 = 5.111V
```
赛题要求5V ± 0.15V（即4.85V~5.15V），5.111V在范围内 ✅

**误差分析**（考虑电阻精度±1%）：
- 最差Vout(高) = 0.6 × (1 + 101k/13.167k) = 0.6 × 8.669 = 5.201V → 5.15V内? **5.201>5.15 ❌**
- 最差Vout(低) = 0.6 × (1 + 99k/13.433k) = 0.6 × 8.369 = 5.021V → 4.85V以上? **5.021>4.85 ✅**

上分压R2用±1%时高限可能超5.15V。建议R2和R3均使用**±0.5%或±0.1%精度电阻**，或选用R2=97.6kΩ+R3=13.0kΩ的组合。
```
97.6k/13.0k → Vout = 0.6×8.508 = 5.105V ✅ 更靠近5.0V
```

**结论**：将当前R2=511kΩ替换为97.6kΩ(±0.5%)，R3=105kΩ替换为13.0kΩ(±0.5%)。

### 2.2 电感选型计算

**依据**：Datasheet 第7.3.2节 Equation (6)
```
L = (Vin - Vout) × Vout / (Vin × fsw × ΔIL)
ΔIL = 20%~40% × IOut_Max = 0.4~0.8A（取0.5A @ 2A输出）
fsw = 500kHz
```

**24V标称输入：**
```
L = (24-5) × 5 / (24 × 500k × 0.5) = 95 / 6M = 15.83μH
```
→ 选 **15μH**（数据手册推荐值，与FXL0630-150-M一致 ✅）

**验证18V最低输入：**
```
D = Vout/Vin = 5/18 = 0.278
ΔIL = (Vin-Vout)×D / (L×fsw) = 13×0.278 / (15μ×500k) = 3.614/7.5 = 0.482A
峰值电流 Ipk = 2A + 0.482/2 = 2.24A
```
FXL0630-150-M饱和电流3.5A > 2.24A ✅

**验证36V最高输入：**
```
D = 5/36 = 0.139
ΔIL = 31×0.139 / (15μ×500k) = 4.309/7.5 = 0.575A
峰值电流 Ipk = 2A + 0.575/2 = 2.29A
```
FXL0630-150-M饱和电流3.5A > 2.29A ✅

**结论**：L=15μH在18~36V宽输入范围内均满足要求。

### 2.5 PT100恒流源方案选取理由及电流计算

#### 方案选取理由

选择 **LM4040A25(2.5V基准) + MCP6004(运放) + MMBT3904(NPN管)** 方案，理由如下：

| 考察方案 | 优点 | 缺点 | 是否选用 |
|---------|------|------|---------|
| **本方案**：基准+运放+NPN | 精度高、温漂低、可调 | 元件数3个 | ✅ 选用 |
| 单一电阻限流 | 最简单、0成本 | 精度差、受温度影响大 | ❌ |
| 恒流二极管(CRD) | 简单 | 温度系数差、难买特定值 | ❌ |
| 专用恒流芯片 | 集成度高 | 成本高、采购周期长 | ❌ |

**选取细节理由**：
1. **LM4040A25**：温漂仅20ppm/°C，初始精度±0.1%，足够工业级测温
2. **MCP6004**：RRIO特性，3.3V供电下输入/输出均可到0~3.3V，不会因轨到轨限制影响恒流
3. **MMBT3904**：NPN通用管，β>100，集电极输出级提供高输出阻抗
4. **1mA选值**：PT100最大电阻157.33Ω时自热功率仅0.157mW，温升<0.01°C

#### 电流计算

```
I = Vref / Rset = 2.5V / 2.5kΩ = 1.000mA ✅
```

**误差分析**：
| 误差源 | 误差值 | 对电流影响 | 对应温度误差 |

| LM4040A25初始±0.1% | ±2.5mV | ±0.1% | ±0.26°C |

满足赛题±10°C基本精度和±8°C优化精度要求 ✅

### 2.6 PT100放大电路增益计算（完整推导）

**赛题要求**：-50°C~+150°C信号在ADC满量程的20%~80%范围内

**已知参数**：
- GD30AD3344差分模式 FSR = ±2.048V
- 20% FSR = 0.4096V, 80% FSR = 1.6384V
- PT100在1mA下电压：80.31mV(-50°C) ~ 157.33mV(+150°C)
- ADC共模偏置：2.5V（由外部LM4040A25提供）
- INA333共模偏置也接2.5V

**增益计算**：
```
目标：让-50°C映射到≈20%FSR，+150°C映射到≈80%FSR

方法：让0°C(100mV)映射到50%FSR(1.024V)
G = 1.024V / 0.1V = 10.24

验证：
-50°C: Vdiff = 80.31mV × 10.24 = 0.822V → 20.1% ✅
  0°C: Vdiff = 100.0mV × 10.24 = 1.024V → 50.0% ✅
+150°C: Vdiff = 157.33mV × 10.24 = 1.611V → 78.7% ✅
```

**INA333增益设置电阻**（Datasheet公式 G = 1 + 100kΩ/Rg）：

#### 基准电压对三个环节的影响

**① 恒流源环节**
```
I = 2.5V / 2.5kΩ = 1mA

```

**② ADC参考电压环节**
```
GD30AD3344使用外部AIN3=2.5V作为参考
ADC输出码 = (Vin / Vref) × 2^23
基准误差→ADC量化误差→温度读数误差
链条：±0.1%基准 → ±0.1%ADC → ±0.3°C
```

**③ INA333共模偏置环节**
```
INA333 REF引脚接2.5V
偏差影响输出共模电压，不影响差分信号→不影响测温
```

### 2.8 低通滤波计算

**前级RC滤波** R7=1.5kΩ, C6=220nF：
```
fc = 1/(2π × 1.5k × 220nF) = 1/(2π × 330μ) = 482Hz

```

**ADC输入端多级滤波** R8/R9=50Ω, C7/C9=100nF/10μF：
```
第一级 fc1 = 1/(2π×50×100nF) = 31.8kHz
第二级 fc2 = 1/(2π×50×10μF) = 318Hz
总50Hz衰减 > -60dB ✅
```
