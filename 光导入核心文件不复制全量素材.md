# 外部素材工作区规则

## 规则

创建外部素材工作区时，遵循**轻量级原则**：

1. **只复制核心即时需要的文件**进入工作区目录（如稿子正文、关键prompt、memory等）
2. **不复制全量素材**（如PPT、附件、备份稿、工具脚本等）
3. 对于未复制的素材，在 `文件清单.md` 中记录其路径和用途
4. 后续需要调用素材时，按原路径按需读取，无需提前占用工作区空间

## 示例

原始目录 `C:\Users\ziming\Desktop\理论宣讲大赛` 有22个文件，工作区内只放3个核心文件 + 1个文件清单：
```
理论宣讲大赛工作区/
├── 宣讲稿正文(双人定稿版).md    # 核心稿子
├── prompt.md                     # 创作规则
├── memory.md                     # 历史记忆
└── 文件清单.md                   # 其他素材索引（按需调用）
