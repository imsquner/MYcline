# multimodal 分类 Prompt：图像理解与多模态处理

## 适用场景
用户提供图像/图片/截图，要求理解其内容、提取信息、分析结构。

## 执行流程

### 1. 识别输入类型
- **截图/UI**：提取界面元素、布局、交互逻辑
- **电路图/原理图**：识别元件、连接关系、信号流向
- **文档扫描件**：OCR识别文字 + 版面分析
- **图表/数据可视化**：提取数值趋势、关键数据点
- **实物照片**：识别物体、场景、特征

### 2. 分析输出规范
```
## 图像分析结果

### 整体描述
[一句话总结图中内容]

### 关键要素
- [要素1]：[描述]
- [要素2]：[描述]

### 提取信息
[结构化呈现提取的数据/文字/关系]

### 结论/建议
[基于图像内容的判断或下一步行动]
```

### 3. 注意事项
- 如果图像质量差（模糊/过暗），先说明限制
- 图表类优先提取具体数值而非定性描述
- 电路图类标注元件标号（R1, C1, U1等）
- 中文文档OCR注意识别简体/繁体和特殊符号
