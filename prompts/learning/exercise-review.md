# 习题整理与解答 Prompt

> 用于课程习题的整理、答案补充和验证
> 适用于：作业整理、习题解答、复习资料生成

## 执行前确认

- 习题来源（课程平台、PDF、Word、PPT等）
- 是否有官方答案可供参考
- 是否需要对答案进行验证

## 核心规则

### 1. 题目范围严格限定
- 只处理作业/习题中实际出现的题目
- **不得擅自添加**其他来源的题目（即使答案正确）
- 如用户要求额外题目，单独标记为"扩展题"

### 2. 题型识别三要素
按以下优先级判断题型：
1. **题型标签**（如"单选题"、"多选题"标签）→ 最可靠
2. **分值信息**（单题分值暗示题型：如2分可能是单选题或多选题）
3. **题目措辞**（"下列正确的是"、"错误的是"等）→ 最不可靠，仅作参考

关键规则：**"正确的是"、"错误的是"等措辞 ≠ 多选题判定依据**

### 3. 答案来源优先级
1. **官方答案**（教师提供的标准答案）
2. **在线测试/考试答案**（已有分数验证的答案）
3. **教材知识点推导**（根据教材内容合理推导）
4. **自行补充**（明确标注"由知识点推导"）

### 4. 模糊措辞处理
- "错误的有_____" → 优先视为单选题，除非明确标注"多选题"
- 如果无法确定题型 → 标注为"题型待确认"
- 宁可遗漏也不要猜测（避免把错题标成多选）

### 5. 输出验证
最终输出必须包含 **"答案对照表"**，标记：
- 每个答案的来源（官方/在线测试/推导）
- 如果来源为推导，标注依据
