## 📋 项目总览

本工作区用于 CIMC 工业嵌入式系统竞赛的**硬件设计部分**，包含 3 块电路板的设计与验证。

### 3 块板子分工

| 编号 | 板子名称 | 核心芯片 | 功能 |
|------|---------|---------|------|
| 01 | **电源板** | GD30DC1354 | DCDC 供电 |
| 02 | **PT100 温度采样板（主板）** | GD30AD3344 | 温度信号调理与数字化采集，SPI 与外部 MCU 通信 |
| 03 | **精密电阻模拟测试板（测试板）** | 精密电阻 | 模拟 PT100 阻值，用于采样板功能验证和精度标定 |
