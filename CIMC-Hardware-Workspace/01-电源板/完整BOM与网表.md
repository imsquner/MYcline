## 四、关键参数汇总

| 参数 | 值 | 来源/说明 |
|------|-----|---------|
| 输入电压 | 24V (18V~36V 宽范围) | 赛题要求 |
| 输出电压 | **5.111V** (标称) | R6=100k, R4=13.3k (DS 官方 5V 设计) |
| 输出电流 | 2A (最大) | GD30DC1354 额定 |
| 开关频率 | 500kHz | DS 固定值 |
| 反馈参考电压 | 0.6V | DS |
| 输出纹波 | ~3.2mV (计算值) | 44.2μF 总输出滤波 |
| UVLO 启动 | ~7.45V | R2=511k, R3=105k |
| UVLO 关断 | ~7.22V | 迟滞 ~0.23V |
| 防反接 Vgs 保护 | **−12V** (齐纳钳位) | D1=MM1Z12V |
| 效率 (估算) | ~89% | 含 P-MOS 压降 |
