# 01 — 电源板：系统供电电源

> **赛题任务**: 基于 GD30DC1354，设计 24V→5V 降压电源板
> **赛题来源**: 2026年CIMC工业嵌入式系统竞赛任务书
> **关联芯片**: GD30DC1354 (GigaDevice)

---

### 1. 电路原理图设计

| # | 任务项 | 状态 | 备注 |
|---|--------|------|------|
| 1.1 | **输入端电路设计** | ✅ | |

| 1.4 | **测量接口设计** | ✅ | 2 针排针 JP1 串联在输出回路，跳线帽短接/断开测电流（见输出端与测量接口设计.md） |

## 设计指标

| 参数 | 要求 | 来源 |
|------|------|------|
| 输入电压（标称） | 24V DC | 赛题任务书 §1 |
| 输入电压（范围） | 18V~36V DC | 赛题任务书 §1.(4) |
| 输出电压 | 5V ± 0.15V | 赛题任务书 §1.(3) |

| 开关频率 | 500kHz | DS 第5页 |
