# 电源板 — PCB 布局指南

> 基于 GD30DC1354 数据手册 + 工程经验
> 帮你一步步画 PCB，每一步告诉你放什么、怎么放、为什么

---

## 一、准备工作

### 1.0 板框尺寸

| 参数 | 值 |
|------|-----|
| **尺寸** | **100mm × 55mm** |
| 形状 | 矩形，四角可加 R2~R3 倒角 |
| 板厚 | 1.6mm（标准） |
| 铜厚 | 1oz（35μm） |

### 1.1 板子层级

| 层 | 用途 |
|----|------|
| **顶层 (Top)** | 放所有元件 + 走功率线（24V / VIN_RAW / SW / VOUT） |
| **底层 (Bottom)** | 大面积 GND 敷铜 + 少量跳线 |

### 1.2 线宽速查（1oz 铜厚）

| 网络 | 电流 | 建议线宽 | 说明 |
|------|------|---------|------|
| 24V / VIN_RAW | ~0.5A | **2mm** | 输入侧，不需要太宽 |
| SW | 2.3A 峰值 | **2~3mm** | 越短越好，< 10mm |
| VOUT | 2A | **3mm** | 输出侧，要够宽 |
| FB / EN / BST | 微量 | **0.3mm** | 信号线，细即可 |
| GND | — | **整片敷铜** | 不靠走线，靠铜皮 |

> 记忆口诀：**功率线 2-3mm，信号线 0.3mm，GND 全铺铜**

## 二、元件放置顺序（按 11 步走）

### 第 1 步：放 U1 (GD30DC1354)

- 放在 PCB 中央偏左的位置
- 方向：Pin1(GND) 朝下，Pin6(BST) 朝上
- 周围留足空间给电容和电阻

```
         ┌──────┐
    BST  │6    5│ EN  ← 左侧
         │      │
    SW   │2    4│ FB  ← 右侧
         │      │
    GND  │1    3│ VIN ← 下侧
         └──────┘
```

### 第 2 步：放 C11 (0.1μF 输入高频去耦) — **最关键的一步**

- 放在 U1 的 **Pin3(VIN) 正下方**
- **距离 < 3mm**（越近越好）
- C11 的 GND 端直接打 2 个过孔到底层 GND 敷铜

```
       U1.3(VIN)
          │
          │ < 3mm
          │
      ┌───┤
      │ C11(0.1μF)
      ├───┤
      │   │
      ╰───╯
        │
   2个过孔到 GND
```

### 第 3 步：放 C1 (0.1μF 自举电容)

- 跨接在 U1.6(BST) 和 U1.2(SW) 之间
- 放在 IC 的 **同一侧**（如上侧）
- 走线越短越好（BST 和 SW 引脚间距本来就小）

```
     C1(100nF)
  ┌───┤├───┐
  │         │
U1.6      U1.2
(BST)     (SW)
```

### 第 4 步：放反馈网络 R6 + R4 + C6

- **紧贴 U1.4(FB) 引脚**
- 三者呈 L 形或直线排列
- 走线总长 < 5mm
- 从 R6 的 VOUT 侧走线到 VOUT 节点

```
        ┌── FB
        │
    R6 ─┤
  (100k)│
        ├── R4(13.3k) ── GND
        │
        └── C6(75pF) ── GND
```

**重要**：FB 网络走线要 **远离 SW 走线和 L1**，至少间距 3mm

### 第 5 步：放 L1 (15μH 功率电感)

- 紧贴 U1.2(SW)
- SW → L1 走线 < 10mm，宽度 2~3mm
- L1 的方向：输入端朝向 U1(SW)，输出端朝向输出电容

```
U1.2(SW) ──── L1 ────→ 到输出电容
         <10mm
        宽 2-3mm
```

### 第 6 步：放输出电容 C8 + C9 + C4 + C10

- **C8(22μF) + C9(22μF)**：紧贴 L1 输出端
- **C4(100nF) + C10(100nF)**：靠近 U2 电流测量排针
- 所有电容的 GND 端通过过孔到底层 GND

```
L1 ──┬── C8(22μF) ── GND
     ├── C9(22μF) ── GND
     ├── C4(100nF) ── GND
     ╰── C10(100nF) ── GND
           │
          VOUT 走线（宽 3mm）
```

### 第 7 步：放 UVLO 电阻 R2 + R3

- 放在 U1.5(EN) 附近
- R2(511k) 一端接 VIN_RAW，另一端接 EN
- R3(105k) 一端接 EN，另一端接 GND
- 走线细即可（0.3mm），因为电流极小

```
VIN_RAW ── R2(511k) ──┬── U1.5(EN)
                       │
                  R3(105k) ── GND
```

### 第 8 步：放 Q1 (IRF9540) + D1 (MM1Z12V)

- Q1 放在 PCB **左侧边缘**（输入侧）
- 三个脚方向：G(左) / D(中) / S(右)
- **D1(齐纳管) 放在 Q1 的 G 和 S 脚之间，距离 < 5mm**
- 齐纳管阴极 → Q1.3(S) → 24V
- 齐纳管阳极 → Q1.1(G) → GND

```
         Q1(IRF9540)
       ╔═══════════╗
  24V──╢ S   D   G ╟──→ VIN_RAW
       ╚═╤═══╤═══╤═╝
         │   │   │
         │   │   ╰──── GND
         │   │
         │   ╰──────── D1(齐纳)阴极
         ╰─────────── D1(齐纳)阳极 → GND

  D1 放这里（紧靠 Q1 的 G 和 S 脚）
```

### 第 9 步：放输入电容 C2 + C3

- **C2(47μF 电解)**：靠近 Q1 漏极（VIN_RAW 节点）
- **C3(1μF 陶瓷)**：在 C2 和 C11 之间
- 三个输入电容按电流路径排：**C2 → C3 → C11 → U1.3(VIN)**

```
24V → Q1 → C2(47μF) → C3(1μF) → C11(0.1μF) → U1.3(VIN)
               ↑                       ↑
         先放大电容再放小电容    高频去耦最靠近 IC
```

### 第 10 步：放外部接口

- **U5(24V 输入端子)**：PCB 左侧边缘
- **LED1 + R5(620Ω)**：放在 PCB 右上角（容易看到的位置）
- **SW2(输出开关)**：PCB 右侧边缘
- **U2(电流测量排针)**：在输出电容和 VOUT 端子之间
- **U3(5V 输出端子)**：PCB 右侧边缘
- **U4(5V 扩展排针)**：PCB 右下
- **U6(24V 扩展排针)**：PCB 左下

```
           ┌──────────────────────────────────┐
    24V  ──┤ U5     U1              LED1 SW2 ├── 5V
    输入   │       GD30DC1354       ●    ○   │   输出
           │                                   │
           │ Q1   C2 C3  L1  C8 C9  U2   U3   │
           │                    测电流   端子  │
           ├──────────────────────────────────┤
    U6 ───┤                                   ├─── U4
   24V扩展 │           GND 敷铜               │  5V扩展
           └──────────────────────────────────┘
```

### 第 11 步：GND 敷铜 + 过孔

- **顶层和底层全铺 GND 敷铜**
- GND 过孔打满：每个电容 GND 端 2 个，IC 周围 4~6 个
- 不要在 GND 敷铜上切出长条空白

## 三、分区原则（画之前想好）

```
┌──────────────────────────────────────────────┐
│               输入区                          │
│   24V → U5 → Q1 → D1 → C2 → C3 → C11        │
├──────────────────────────────────────────────┤
│                核心区                         │
│      U1(GD30DC1354) + L1 + 反馈电阻          │
│      ↑ 这是全板最关键的 2cm²                  │
├──────────────────────────────────────────────┤
│               输出区                          │
│   L1 → C8/C9 → U2 → SW2 → U3/VOUT           │
│        + C4/C10 + LED1 + R5                  │
├──────────────────────────────────────────────┤
│             扩展区                            │
│   U4(5V扩展) + U6(24V扩展)                   │
└──────────────────────────────────────────────┘
```

**电流走向：从左到右**（24V 输入从左边进，5V 输出从右边出）

## 四、10 条必须遵守的铁律

| # | 规则 | 不遵守的后果 |
|---|------|------------|
| 1 | **C11(0.1μF) 必须紧贴 U1.3(VIN)，距离 < 3mm** | 输入噪声大，输出纹波翻倍 |
| 2 | **SW 走线 < 10mm，远离 FB 网络** | 开关噪声耦合到反馈，输出抖动 |
| 3 | **R6/R4/C6 紧贴 U1.4(FB)，走线 < 5mm** | FB 受干扰，输出不准 |
| 4 | **C1(100nF) 紧贴 U1.6(BST) 和 U1.2(SW)** | 自举不足，高侧 MOS 驱动不良 |
| 5 | **功率走线宽度：24V=2mm, SW=2-3mm, VOUT=3mm** | 线细了压降大、发热 |
| 6 | **输入/输出 GND 分开回流，单点汇合** | 地弹噪声，环路不稳定 |
| 7 | **D1(齐纳管) 紧靠 Q1 的 G 和 S 脚 < 5mm** | Vgs 保护失效，MOS 管损坏 |
| 8 | **FB 走线用地线包围（Guard Ring）** | FB 受干扰，输出不准 |
| 9 | **SW 节点下方禁止走任何信号线** | 信号被 SW 的 dV/dt 干扰 |
| 10 | **GND 过孔越多越好** | 回流阻抗高，噪声大 |

## 五、放好元件后的走线顺序

```
第 1 步：走 24V 输入线（U5 → Q1.3）
第 2 步：走 VIN_RAW 线（Q1.2 → C2/C3/C11 → U1.3）
第 3 步：走 SW 线（U1.2 → L1）← 最短最宽
第 4 步：走 VOUT 线（L1 → C8/C9 → U2 → SW2 → U3）
第 5 步：走 FB 网络（U1.4 → R6 → R4 → C6）
第 6 步：走 EN 网络（U1.5 → R2 → R3）
第 7 步：走 LED 网络（VOUT → R5 → LED1 → GND）
第 8 步：走 5V 网络（SW2 → U3/U4）
第 9 步：走 24V 扩展（U5 → U6）
第10 步：GND 敷铜 + 打过孔
```

## 六、画完后自查清单

放好元件、走完线后，逐项打勾：

- [ ] C11 距离 U1.3(VIN) < 3mm？
- [ ] SW 走线 < 10mm？
- [ ] SW 走线与 FB 网络间距 ≥ 3mm？
- [ ] R6/R4/C6 在 FB 脚旁边 < 5mm？
- [ ] C1 紧贴 BST 和 SW 脚？
- [ ] 24V 走线 ≥ 2mm？
- [ ] VOUT 走线 ≥ 3mm？
- [ ] D1 紧靠 Q1 的 G 和 S 脚？
- [ ] 输入电容 GND 和输出电容 GND 分开回流？
- [ ] GND 过孔够多？
- [ ] 24V 输入从左边进，5V 输出从右边出？
- [ ] 所有元件都有对应的封装，没有漏焊盘？
