# 电源板 — 设计笔记（最终版）

> 24V→5V/2A 同步 Buck 电源板，基于 GD30DC1354。
> 本文档记录设计过程中的关键决策、参数计算和最终版本。

---

## 设计概述

| 项目 | 内容 |
|------|------|
| 芯片 | GD30DC1354 (同步降压 DC/DC) |
| 输入 | **24V DC** (18V~36V 宽范围) |
| 输出 | **5V DC** / 2A max |
| 开关频率 | 500kHz (固定) |
| 防反接 | IRF9540 P-MOSFET + 12V 齐纳管保护 |

## 设计过程与关键决策

### 1. 核心方案：GD30DC1354 参考设计

**来源**：数据手册 Figure 4 (第10页)，官方 5V/2A 参考电路。

原参考电路元件：
- L1 = 15μH, C1 = 10μF, C2 = 0.1μF, C3 = 0.1μF, C4 = 22μF×2
- R1 = 100kΩ, R2 = 13.3kΩ
- R4 = 511kΩ, R5 = 105kΩ (UVLO)

### 2. 输入滤波增强

**变更**：将 C1 (10μF 陶瓷) → **C2 = 47μF 铝电解** + **C3 = 1μF 陶瓷** + C11 = 0.1μF

**原因**：赛题要求"电解+陶瓷"并联输入滤波。47μF 电解提供低频储能，比 10μF 陶瓷更适合大电流输入。

### 3. 防反接保护 — 从肖特基改为 P-MOSFET

**初始方案**：SS56 肖特基二极管 (5A/60V)

**最终方案**：**IRF9540 P-MOSFET 单管** (源极接输入正极，漏极接后级，栅极通过 12V 齐纳管接源极)

**变更原因**：肖特基在大电流下压降 0.4~0.5V、发热严重 (满载约 1W)。P-MOSFET Rds≈0.2Ω，压降仅 ~0.1V，效率提升。

**重要发现 — Vgs 保护**：P-MOS 栅极直接接 GND 时，Vgs = −24V，超出 IRF9540 的 ±20V 最大额定值。解决方案：**增加 D1 (MM1Z12V 12V 齐纳管)**，将 Vgs 钳位在 −12V，远低于 ±20V。

### 4. 反馈电阻确认

**来源**：数据手册第11页 Table 3，5V 输出推荐 R1=100kΩ, R2=**13.3kΩ**

```
VOUT = 0.6 × (1 + 100/13.3) = 5.111V
```

**备注**：最初文档中提到 13.7kΩ (VOUT=4.979V)，但查数据手册后确认 13.3kΩ 是官方推荐值，无需修改。

### 5. 保险丝移除

**最终决定**：不带保险丝。24V 直入 → P-MOS 防反接 → 后级。

### 6. UVLO 使能设置

```
EN 分压：R2(511k) + R3(105k)
UVLO 启动 = 1.27 × (511+105)/105 ≈ 7.45V
UVLO 关断 = 1.23 × (511+105)/105 ≈ 7.22V
```

远低于 18V 最低输入，不影响正常工作。

### 7. 输出电容增强

DS 推荐 2×22μF 陶瓷。实际增加 2×100nF 高频去耦：
- C8 + C9 = 22μF × 2 (1210)
- C4 + C10 = 100nF × 2 (0603)
- **总输出滤波 = 44.2μF**，纹波仅 ~3.2mV

### 8. 功能扩展

- **SW2**：滑动开关，控制 5V 输出通断 (正极回路)
- **U2**：2 针排针串联 VOUT，拔下跳线帽可串电流表测输出电流
- **LED1+R5(620Ω)**：输出指示，电流 ~4.8mA
- **U4(5V×2+GND×2) + U6(24V×2+GND×2)**：扩展接口，给外部模块供电

## 最终电路参数计算汇总

### 输出电压

```
VOUT = Vref × (1 + R6/R4) = 0.6 × (1 + 100/13.3) = 5.111V
```

### 电感纹波电流

```
ΔIL = VOUT × (1 − VOUT/VIN) / (fsw × L)
@24V: ΔIL = 5 × (1−5/24) / (500k × 15μ) = 0.528A (26.4%)
@36V: ΔIL = 5 × (1−5/36) / 7.5 = 0.574A (28.7%)
峰值 = 2A + 0.574/2 = 2.287A < 3A ISAT ✅
```

### 输出纹波

```
Vripple_c = ΔIL / (8 × fsw × Cout) = 0.528 / (8×500k×44.2μ) = 2.99mV
Vripple_esr ≈ 1.06mV
总纹波 ≈ 3.17mV ✅
```

### 效率估算

```
P-MOS 压降: 0.46A × 0.2Ω = 0.09V
U1 损耗 (Rdson): 2A² × (0.15+0.078) × 0.5 ≈ 0.45W
输出功率: 10W
效率 ≈ 10 / (10 + 0.09×0.46 + 0.45 + 其他) ≈ 89%
```

## PCB 布局要点

1. 输入电容 C2→C3→C11 从大到小靠近 U1.3 (VIN)
2. C1 (自举) 尽量靠近 U1 的 BST 和 SW 脚
3. L1 靠近 SW 脚，走线短而宽
4. 反馈分压 R6/R4/C6 靠近 U1.4 (FB)
5. SW 节点走线远离 FB 网络
6. Q1 (TO-220) 插件区与贴片区保持间距
7. 功率地 (GND) 用星形接地或大面积覆铜
