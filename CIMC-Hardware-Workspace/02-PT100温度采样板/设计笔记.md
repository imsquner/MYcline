# PT100 温度采样板 — 设计笔记

> 记录 PT100 温度采样板设计过程中的方案、计算、选型、决策等。

## 设计概要

| 项目 | 内容 |
|------|------|
| 核心芯片 | GD30AD3344（24位 Σ-Δ ADC） |
| 传感器 | PT100 铂电阻温度传感器（-50°C ~ 200°C） |
| 通信接口 | SPI（与外部 MCU 通信） |
| 供电 | 通过接口从电源板取电（+5V / ±5V 待确认） |

## 功能框图

```
PT100 → 信号调理电路 → GD30AD3344 ADC → SPI → 外部 MCU
                          ↑
                    参考电压源
```
