# 🔍 信息溯源规则

> 本工作区所有技术信息必须可追溯至原始资料。
> 适用于 AI 助手与用户之间的所有设计讨论。

---

## 核心原则

**所有涉及以下内容的信息，必须标注出处：**

1. **芯片参数** — 引脚定义、电气特性、时序等 → 引用数据手册（PDF 页码/章节）
2. **电路参数** — 电阻电容值、增益、带宽等 → 引用计算过程或数据手册推荐值
3. **技术名词** — 首次出现时给出定义 + 出处
4. **赛题要求** — 引用赛题 PDF 的具体条目

## 引用格式

```
[来源: 文件名 → 章节/页码]
```

**示例：**
```
GD30DC1354 的开关频率为 1.2MHz
[来源: GD30DC1354_Datasheet_Rev1.0.pdf → 第6页 Table 3]
```

## 不可编造条款

- ❌ 无法在源文件中找到的信息 → 不得编造
- ✅ 标注「⚠️ 未验证/待查证」并说明缺失原因
- ✅ 主动追问用户是否可提供缺失的资料
