# 03 — 精密电阻模拟测试板（测试板）

> **核心元件**: 精密电阻网络
> **功能**: 通过板载精密电阻模拟不同温度下 PT100 传感器的阻值，用于对采样板进行功能验证和精度标定，替代昂贵的标准温度源设备

## 设计目标

- [ ] 确定模拟温度点（覆盖 -50°C ~ 200°C 的典型阻值）
- [ ] 精密电阻选型（精度 ≥ 0.1%，温漂低）
- [ ] 切换电路设计（拨码开关/跳线帽/继电器）
- [ ] 接口定义（与采样板对接的排针/排线）
- [ ] 布局布线要点

## PT100 阻值-温度参考
