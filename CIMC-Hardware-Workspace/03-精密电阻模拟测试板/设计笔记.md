# 精密电阻模拟测试板 — 设计笔记

> 记录精密电阻模拟测试板设计过程中的方案、计算、选型、决策等。

## 设计概要

| 项目 | 内容 |
|------|------|
| 用途 | 模拟 PT100 在不同温度下的阻值，用于采样板标定 |
| 核心元件 | 精密电阻网络 |
| 切换方式 | 待定（拨码开关 / 跳线帽 / 继电器 / 模拟开关） |
| 对接接口 | 与 PT100 采样板的传感器接口匹配（2线/3线/4线？） |

## 设计思路

### 方案A：拨码开关 + 固定电阻
- 每个温度点对应一路精密电阻
- 通过拨码开关选择接入哪一路
- 优点：简单、成本低
- 缺点：需要手动切换，点数有限

### 方案B：继电器网络
- 用继电器切换不同阻值
- 可通过 GPIO 控制，实现自动化测试
- 优点：可程控
- 缺点：体积大、成本高

### 方案C：精密电阻 + 跳线帽
- 板子上焊好一排精密电阻，用跳线帽短接选择
- 最简单，适合初版验证
