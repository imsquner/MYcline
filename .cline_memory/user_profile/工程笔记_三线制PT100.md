# 三线制PT100接线定义（赛题原文）

> 赛题：三线制PT100传感器引脚定义
> 来源：CIMC/华为嵌入式竞赛赛题原文

**定义：**
- **白线**：电流激励，同时检测PT100电阻值变化对应的电压信号，接INA333 VIN+
- **红线A**：电流回路（返回GND），承载恒流源返回电流
- **红线B**：引线电阻补偿，接INA333 VIN−（不承载电流）

**引线电阻抵消原理：**
两根红线等长等材质，导线电阻相同。白线承载激励电流至PT100，INA333 VIN+检测白线端电压；红线B引至INA333 VIN−，不承载电流，仅检测PT100另一端的电压。差分测量V(VIN+)−V(VIN−)中，两根红线上各自的导线压降（白线激励电流×红线电阻 vs 红线B零电流）被差分放大器抵消，仅剩PT100本体的阻值压降。

**⚠ 常见错误（我反复犯的）：**
- ❌ 红线激励 + 白线补偿 + 黑线回路 — **错误！赛题没有黑线**
- ✅ 白线激励 + 红线A回路 + 红线B补偿 — **正确**

**对应U6端子（原理图）：**
- U6.1 → 红线B（补偿/检测-）
- U6.2 → 白线（激励+检测+）
- U6.3 → 红线A（电流返回）
